# QR_code-generator This project is a simple Python QR Code Generator that creates a QR code from any text, link, or data and saves it as an image file.

It uses the qrcode Python library to generate QR codes quickly and easily.

QR codes can be used for:

Website links

Contact information

WiFi passwords

Payment links

Any text-based data

⚙️ Features

Generate QR codes from any text or URL

Save QR code as a PNG image

Customizable file name

Adjustable QR code size and border

Beginner-friendly Python code

🧰 Technologies Used

Python

qrcode library

Pillow (PIL) – used internally for image generation

📦 Installation

Install the required library using pip:

pip install qrcode[pil]
